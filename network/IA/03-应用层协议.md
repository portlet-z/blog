## 概述

- TCP/IP模型的最高层
- 直接为应用程序提供网络服务

## DNS

- Domain Name System, 域名解析系统
- 建立IP地址与域名之间的映射关系
- 将域名解析为IP地址
- 将IP地址解析为域名

| 命令                 | 备注        |
| -------------------- | ----------- |
| nslookup             | DNS调试工具 |
| ipconfig /displaydns | 查看DNS缓存 |
| ipconfig /flushdns   | 清空DNS缓存 |
